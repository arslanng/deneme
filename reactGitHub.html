Bir React Uygulamasını GitHub Sayfalarında Kullanmak

Repoya projeyi eklemek.

React projesi bir github reposuna eklenir.

Terminale npm install gh-pages --save-dev yazılarak gerekli paket yüklenir.

package.json dosyasına 

{
    "name": "my-app",
    "version": "0.1.0",
    "homepage": "https://gitname.github.io/react-gh-pages",
    "private": true,
altı çizili kısım eklenir.

package.json dosyasına 

{
    "scripts": {
    "predeploy": "npm run build",
    "deploy": "gh-pages -d build",
    "start": "react-scripts start",
    "build": "react-scripts build",
altı çizili kısım eklenir.

Terminale npm run deploy yazılarak deploy işlemi yapılır.

deploy işlemine mesaj eklenmesi için npm run deploy -- -m "Deploy React app to GitHub Pages" kalıbı kullanılır.

Repoya Deploy Edilen Projeyi Yayınlamak

İlgili reponun github sayfasında settings tıklanır.

settings altında Code and automation > Pages tıklanır.

    Bu sayfada aşağıdaki ayarlar yapılır.
  • Source: Deploy from a branch
  • Branch:
    • Branch: gh-pages
    • Folder: / (root)

Save edilir.

Yayınlanan sayfa: kullaniciadi.github.io/repoAdi
